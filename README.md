# baekjoon solution

* src 폴더에 각 문제의 소스 코드가 있다.
    - 파일 이름은  '_문제번호_문제이름_난이도' 구성을 갖고 있다.
    - 문제 번호대로 https://www.acmicpc.net/ 에서 검색하면 문제를 보거나 테스트할 수 있다.
    - 문제 이름은 그대로 이거나 내 마음대로 한영 번역한 것이다.
    - 난이도는 기본적으로 1-5순으로 어렵다. 체감 난이도이므로 모두에게 적용될 난이도는 아니다.
    - 단, 코드그라운드의 난이도와 유사하게 매겨졌다. 1은 EASY 2는 NORMAL1 3은 NORMAL2 4는 HARD 5는 VICTORIA급이지만 정확하진 않다.
    - 퍼알 이름 뒤에 _Fail이 붙은 것은 사이트에서 테스트를 통과하지 못한 답안이다.
